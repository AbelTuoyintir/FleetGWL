## 2025-07-11 - [Global Clipboard Utility & Monospace Alphanumerics]
**Learning:** Alphanumeric identifiers like License Numbers and License Plates are prone to character confusion (e.g., '0' vs 'O'). Using monospace fonts (`font-mono`) and providing immediate copy-to-clipboard functionality significantly reduces user error and frustration during data entry and verification tasks. Globalizing these utilities ensures consistent behavior and UI feedback (toasts) across the entire application.
**Action:** Always wrap alphanumeric identifiers in `font-mono` and provide a copy button using the global `copyToClipboard` utility for precision data fields.
